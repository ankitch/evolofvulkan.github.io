WHAT IS VULKAN API?

Vulkan is a new generation graphics and compute API that provides high-efficiency, cross-platform access to modern GPUs used in a wide variety of devices from PCs and consoles to mobile phones and embedded platforms.

Open Source

Vulkan is completely open-source. Its code base can be easily acquired in github. With many active open-source contributors out there, Vulkan's prospect is very promising.

Cross Platform

Vulkan is cross-platform. It can work across a wide range of hardwares—Intel, AMD, NVIDIA, including mobile platforms. Vulkan can be used in many operating systems like linux, OSX, and Windows.

Adopted By Android

Vulkan is officially adopted by Android. Several android devices like Samsung's latest phones Galaxy S7 and S7 edge, NVIDIA Android TV, and NVIDIA Shield Tablet K1 are already boarding the Vulkan API.

Supported By Many Companies

Vulkan has been a huge attraction to many reputable companies. Tech Giants like Google, Intel, AMD, NVIDIA, Samsung, etc. are supporing Vulkan.
