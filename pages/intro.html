Introduction

<<<<<<< HEAD

Vulkan is a low-level API for 3D graphics and compute applications, which was brought to us by Khronos Group. Low-level abstraction simply means it is more close to the hardware allowing the applications to have more control on how they handle the hardware, and hence increasing the performance if programmed well.
